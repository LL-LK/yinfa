# API接口文档

## 基础信息

- 基础URL: `http://localhost:8000/api` (本地) 或 `https://your-railway-url/api` (生产)
- 请求方式: GET/POST
- 响应格式: JSON
- 响应结构:
  ```json
  {
    "code": 0,
    "data": {},
    "error": ""
  }
  ```

## 系统接口

### 健康检查

**接口**: `GET /health`

**响应示例**:
```json
{
  "status": "ok",
  "timestamp": "2026-04-29T10:00:00.000Z"
}
```

### 服务器信息

**接口**: `GET /`

**响应示例**:
```json
{
  "message": "Travel Mini Program API Server",
  "version": "1.0.0",
  "endpoints": {
    "health": "/health",
    "api": "/api"
  }
}
```

## 通用错误响应

所有接口在发生错误时返回统一格式：

### 400 参数错误
```json
{
  "code": 1,
  "error": "参数验证失败: openid: openid不能为空"
}
```

### 401 未授权
```json
{
  "code": 1,
  "error": "未登录或登录已过期"
}
```

### 404 资源不存在
```json
{
  "code": 1,
  "error": "Product not found"
}
```

### 429 请求过于频繁
```json
{
  "code": 1,
  "error": "请求过于频繁，请稍后再试"
}
```

### 500 服务器错误
```json
{
  "code": 1,
  "error": "服务器内部错误，请稍后重试"
}
```

}

### 11. 获取天气信息

**接口**: `GET /weather`

**参数**:
- `city`: 城市名称（可选，默认为"桂林"）

**支持的城市**: 桂林、北京、上海、广州、深圳、成都、杭州、西安

**响应示例**:
```json
{
  "code": 0,
  "data": {
    "condition": "晴",
    "temp": "25°C",
    "humidity": "60%",
    "wind": "东北风2级",
    "city": "桂林",
    "reportTime": "2026-04-29 10:00:00"
  }
}
```

**说明**: 需要配置 `AMAP_KEY` 环境变量调用高德天气API，未配置时返回模拟数据。

**错误响应**:
- `500`: `{ "code": 1, "error": "Failed to fetch weather" }`
