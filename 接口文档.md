# API接口文档

## 基础信息

- 基础URL: `http://localhost:8000/api`
- 请求方式: GET/POST
- 响应格式: JSON
- 响应结构:
  ```json
  {
    "code": 0, // 0表示成功，1表示失败
    "data": {}, // 成功时返回的数据
    "error": "" // 失败时返回的错误信息
  }
  ```

## 接口列表

### 1. 获取分类列表

**接口**: `GET /categories`

**响应示例**:
```json
{
  "code": 0,
  "data": [
    {
      "id": 1,
      "name": "景点门票",
      "slug": "tickets",
      "order": 1
    },
    {
      "id": 2,
      "name": "酒店住宿",
      "slug": "hotels",
      "order": 2
    }
  ]
}
```

### 2. 获取商品列表

**接口**: `GET /products`

**参数**:
- `category`: 分类slug（可选）
- `search`: 搜索关键词（可选）

**响应示例**:
```json
{
  "code": 0,
  "data": [
    {
      "id": 1,
      "name": "故宫门票",
      "description": "北京故宫成人票",
      "price": 60,
      "stock": 100,
      "image_url": "/src/assets/image/1.jpg",
      "is_active": true,
      "category": {
        "id": 1,
        "name": "景点门票",
        "slug": "tickets"
      }
    }
  ]
}
```

### 4. 用户登录

**接口**: `POST /users/login`

**请求体**:
```json
{
  "openid": "用户的openid",
  "nickname": "用户昵称",
  "avatar_url": "用户头像URL",
  "phone": "用户手机号"
}
```

**响应示例**:
```json
{
  "code": 0,
  "data": {
    "id": 1,
    "openid": "用户的openid",
    "nickname": "用户昵称",
    "avatar_url": "用户头像URL",
    "phone": "用户手机号",
    "created_at": "2026-04-27T10:00:00Z"
  }
}
```

### 5. 获取用户详情

### 6. 创建地址

**接口**: `POST /address/create`

**请求体**:
```json
{
  "openid": "用户的openid",
  "full_name": "收货人姓名",
  "phone": "联系电话",
  "address_line": "详细地址",
  "city": "城市",
  "postal_code": "邮政编码",
  "is_default": true
}
```

**响应示例**:
```json
{
  "code": 0,
  "data": {
    "id": 1,
    "full_name": "收货人姓名",
    "phone": "联系电话",
    "address_line": "详细地址",
    "city": "城市",
    "postal_code": "邮政编码",
    "is_default": true
  }
}
```

### 8. 创建订单

**接口**: `POST /order/create`

**请求体**:
```json
{
  "openid": "用户的openid",
  "items": [

      "product_id": 1,
      "quantity": 2
    }
  ]
}
```

**响应示例**:
```json
{

  "data": {
    "id": 1,
    "order_no": "ORD1234567890123",
    "total_price": 120,
    "status": "pending",
    "created_at": "2026-04-27T10:00:00Z",
    "updated_at": "2026-04-27T10:00:00Z",
    "items": [
      {
        "id": 1,
        "product": {
          "id": 1,
          "name": "故宫门票",
          "description": "北京故宫成人票",
          "image_url": "/src/assets/image/1.jpg"
        },
        "quantity": 2,
        "unit_price": 60,

  }
}
```
